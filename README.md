# プログラミング実習課題

# ##　概要

=================================================================  
node.jsを使用して、CLIタスク管理ツール「my-todo」を作成する

## セットアップ手順

=================================================================  
1 対象のパッケージをインストールする
npm install commander dayjs chalk@4 uuid@8
2「.gitignore」 ファイルを作成し、「node_modules/」を追加する
3 package.jsonに「 "type": "module",」を追加する

## 各コマンドの使用方法

=================================================================  
node todo.js add "タスク名" ⇒TODOタスクを「tasks.json」に追加する タスク名：タスク名称を記入  
node todo.js list ⇒「tasks.json」に保存されたTODOタスクのリストを表示する  
node todo.js done "ID"⇒「tasks.json」に保存されたTODOタスクのステータスを「未完了」⇒「完了」に更新する ID:「tasks.json」に保存されているIDを指定  
node todo.js delete "ID" ⇒指定されたIDのタスクを「tasks.json」から削除する ID:「tasks.json」に保存されているIDを指定

## 使用したライブラリ一覧

==================================================================  
commander :node.jsコマンドライン インターフェース  
chalk :ターミナル文字列のスタイル設定を行う (V.5.0はESM専用となっているので、今回はV.4.0を使用)  
dayjs :日付と時刻を解析、検証、操作、表示する  
uuid :一意なIDを生成 (V.13.0はcommonjsに対応されていない為、今回はV.8.0を使用)
